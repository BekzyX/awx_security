Vulnerability Exploits:

SQL Injection

SQL Injection is a code injection technique that exploits vulnerabilities in an application's software by manipulating SQL queries. 

                    
SELECT * FROM users WHERE username = 'admin'--' AND password = 'password';

Cross-Site Scripting (XSS)

XSS is a security vulnerability that allows an attacker to inject malicious scripts into content from otherwise trusted websites. 

                    
<script>alert('XSS')</script>
